To ensure a high quality of the system, an iterative development process has been striven for, wherein the functionality and structure have been redesigned multiple times. 
\par
However, no distinct iterations have been made and as a result the development process has only been loosely iterative, with the workflow shifting from iteration to iteration in a less definable way. In the report only the final iteration of the project has been presented.
\par
Thanks to Morten Rom, Kasper Andersen, and Aalborg Zoo for their cooperation and to supervisor Lu Chen for her support during the project. 
\par
The program that has been developed in this project can be found on GitHub with this link: https://github.com/tlorentzen/P3
